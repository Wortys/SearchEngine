# Дипломный проект: SearchEngine

Поисковый движок для корпоративного портала (аналог упрощенного поиска). Программа индексирует текстовые файлы и ищет по ним запросы, выдавая самые подходящие результаты.

## Что использовал (стек)
- C++ 17
- CMake (сборка)
- nlohmann_json (для работы с конфигами)
- Google Test (юнит-тесты)
- Многопоточность (std::thread)

## Как запустить проект
Для работы нужны файлы config.json и requests.json в папке с программой.

1. Создать папку build: `mkdir build && cd build`
2. Сконфигурировать: `cmake ..`
3. Собрать: `cmake --build .`
4. Запустить: `./SearchEngine` (или .exe на Windows)

После запуска появится файл answers.json с результатами поиска.
