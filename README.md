# Search Engine

Корпоративный поисковый движок. Консольное приложение, которое индексирует файлы и осуществляет поиск по ним с учетом релевантности.

## Стек технологий
* **C++17**
* **CMake** — система сборки
* **nlohmann/json** — работа с JSON
* **Google Test** — модульное тестирование
* **Multithreading** — многопоточная индексация документов

## Как запустить
1.  Убедитесь, что установлен **CMake** и компилятор C++ (MinGW или MSVC).
2.  Рядом с исполняемым файлом должны лежать:
    * Файл конфигурации `config.json`.
    * Файл запросов `requests.json`.
    * Папка `resources` с текстовыми файлами для поиска.
3.  Сборка проекта:
    ```bash
    mkdir build
    cd build
    cmake ..
    cmake --build .
    ```
4.  Запуск:
    ```bash
    ./SearchEngine
    ```
    Результат работы будет записан в файл `answers.json`.

## Тестирование
Для запуска тестов выберите цель `search_engine_test` при сборке и запустите полученный исполняемый файл.
